# 🎬 Sakib Salim — Portfolio

A modern, Apple-inspired personal portfolio website showcasing video editing, motion design, and creative work.

---

## 🚀 Overview

This portfolio highlights my work as a **Video Editor & Visualizer**, featuring:

- 🎥 Video editing projects (Long-form & Short-form)
- 🎨 Clean, minimal UI design
- 📄 Dynamic resume page
- ⚡ Smooth interactions and responsive layout

---

## ✨ Features

- 🔥 Apple-style glass UI (blur + glow effects)
- 🎬 Video portfolio with modal playback
- 🧩 Toggle system (Long-form / Short-form)
- 📱 Fully responsive design
- 📄 Dynamic resume (data-driven)
- 🧠 Clean component-based structure
- ⚡ Fast build using Vite

---

## 🛠 Tech Stack

- HTML5  
- CSS3 (TailwindCSS)  
- JavaScript (ES Modules)  
- Vite
